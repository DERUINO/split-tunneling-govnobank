### Requirements
- skype for business 2016
- telegram desktop
- vk teams

### Запуск
- Заполните файл `interface_ip.txt`:
Введите ip сетевого интерфейса, на который будут проксироваться запросы вышеуказанных приложений

    example:
    `192.168.1.123`

- Запустите от имени администратора `split.bat`
